## 🎯 출제 의도 핵심 요약

### FSKU는 다음 역량을 평가하고자 함:

1. **법규 준수 능력** (52%)
   - 금융 관련 법령 조항 숙지
   - 규정 위반 시 처벌 내용 이해
   - 법적 요구사항 실무 적용

2. **기술적 보안 지식** (35%)
   - 네트워크/시스템 보안 이해
   - 암호화 기술 적용 능력
   - 최신 보안 위협 인지

3. **실무 적용 능력** (24%)
   - 시나리오 기반 문제 해결
   - 우선순위 판단
   - 대응 방안 수립

4. **개념 이해도** (22%)
   - 금융/보안 기본 용어
   - 핵심 개념 정의
   - 원리 이해
